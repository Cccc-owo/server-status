## 介绍

友情时光是一个横跨1.7.X-1.16.5的小游戏服务器，原立项日期是21年，实际发布时间是22年6月29日，目前服务器已有的玩法包括起床战争，空岛战争，职业战争，搭路小游戏和饥饿游戏，后续还会增加更多玩法，当然，这一切都离不开各位玩家们的支持！
